# Student Registration Form

## Project Description
This is a **Student Registration Form** created using **HTML, CSS, and JavaScript**.  
The form collects the following information from the student:

- Full Name  
- Roll Number  
- Department  
- Gender  
- Contact Number  

It also includes **basic form validation** using JavaScript to ensure all fields are filled and contact number is valid.

---

## Features
- Responsive and styled form using CSS  
- Category selection using dropdown (Department)  
- Gender selection using radio buttons  
- Contact validation (10-digit number)  
- Success and error messages on submission  

---

## How to Use
1. Open `index.html` in any web browser.  
2. Fill in all the required fields: Name, Roll Number, Department, Gender, and Contact Number.  
3. Click the **Register** button.  
4. If all fields are valid, a success message will appear.  
5. If any field is missing or invalid, an error message will appear.  

---

## Technologies Used
- **HTML5** – Structure of the form  
- **CSS3** – Styling and layout  
- **JavaScript** – Form validation and dynamic feedback
